## 📖 產品定義 (What is this?)

**蛍光ペン PRO** 是一款結合日系美學與硬派功能的 **SEO 關鍵字螢光筆 (SEO Keyword Highlighter)**。

不同於傳統編輯器，它專注於解決 SEO 寫手在「撰稿 -> 標註 -> 交付」過程中的斷層。透過獨家的格式封裝技術，讓您在標註 **H1/H2 標籤** 與 **紅字重點** 後，能完美無縫地將內容轉移至 **Google Docs**，並自動套用 **微軟正黑體**，徹底告別格式跑版的惡夢。

## ⚡ 痛點與解決方案 (Pain & Solution)

我們針對 SEO 寫手最常遇到的四大痛點，提供了一站式解決方案：

| 文案困擾 (Pain Points) | 蛍光ペン PRO 的解法 (Solution) |
| :--- | :--- |
| **Google Docs 格式災難** | 獨家 **Format Injection 技術**，複製時自動轉為適合中文閱讀的 **微軟正黑體**，並鎖定紅字/粗體樣式不丟失。 |
| **關鍵字堆砌風險** | 內建 **關鍵字密度檢測 (Density Monitor)**，即時計算百分比，避免因過度優化 (Keyword Stuffing) 遭演算法懲罰。 |
| **手動標註耗時** | 提供 **一鍵標註 (Auto Mark)** 功能，瞬間將所有目標關鍵字標示為 **[朱色+粗體]**，視覺化關鍵字分佈。 |
| **標題層級混亂** | 具備 **H1/H2/P 結構化按鈕** (對應 24pt/18pt)，直觀設定 HTML 語意標籤，提升 SEO 架構分數。 |

## ✨ 核心機能 (Key Features)

### 1. 沉浸式寫作環境 (Zen Workspace)
* **日系雜誌排版**：採用 `Noto Serif JP` (明朝體) 與 `Noto Sans JP`，提供如紙本般的閱讀體驗。
* **深海夜間模式**：支援 Dark Mode 切換，減少長時間工作的視覺疲勞。
* **全裝置響應**：無論是桌機雙欄模式，或手機/平板的分頁模式 (Tabs)，排版永遠完美適應。

### 2. SEO 智能輔助 (SEO Intelligence)
* **即時密度監控**：在側邊欄即時顯示關鍵字出現次數與密度 (Density %)。
* **無損字級技術**：標註過程僅改變顏色權重，**絕對不影響** 原本設定的字體大小 (Font Size Agnostic)。
* **全形標點轉換**：內建轉換開關，一鍵修正半形符號，符合專業出版規範。

### 3. 無縫交付 (Seamless Delivery)
* **一鍵複製到 Docs**：按鈕整合了格式清洗邏輯，直接貼上 Google Docs 即可交付，無需二次排版。
* **純文字備份**：支援 `.txt` 格式導出，方便原始碼保存。

## 📝 使用教學 (User Guide)

只需四個步驟，完成一篇標準化的 SEO 文章：

1.  **【入稿】Input**
    * 在左側「原文輸入」貼上草稿，開啟「全形標點」自動修正格式。
2.  **【定義】Define**
    * 在「關鍵字庫」輸入目標 SEO 關鍵字（每行一個）。
    * 點擊 `一鍵標註`，系統將自動為您畫上重點。
3.  **【結構】Structure**
    * 選取大標題，點擊上方工具列的 `H1` (24pt)。
    * 選取副標題，點擊 `H2` (18pt)。
    * 確保文章結構層級分明，提升易讀性。
4.  **【交付】Export**
    * 點擊右下角的 `複製到 Docs` (支援微軟正黑體)。
    * 前往 Google Docs 按下貼上 (Ctrl+V)，格式完美呈現。
