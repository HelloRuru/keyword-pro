## 📖 產品定義 (Definition)

**蛍光ペン PRO (Highlighter PRO)** 是一款運行於瀏覽器的 **SEO 寫作輔助工具 (SPA)**。它結合了傳統日式文具的侘寂 (Wabi-sabi) 美學與現代硬派 SEO 功能。

本工具專注於解決專業寫手在「撰稿、標註、交付」流程中的斷層，特別是將富文本內容轉移至 **Google Docs** 或跨境發布時的格式兼容性問題。透過獨家的樣式封裝技術，讓您的文章在交付時依然保持完美的結構與視覺重點。

## ✨ 核心機能 (Key Features)

### 1. 利休色調・沉浸式寫作空間 (Rikyu Workspace)
* **和風紙質感**：以「白橡色」為基底的溫潤背景，搭配「油色」墨跡，提供如高級和紙般的書寫體驗。
* **視覺引導設計 (Jump Colors)**：
    * **[茜色]** 引導「一鍵標註」與「A+ 重點」。
    * **[紺青色]** 引導「複製交付」。
    * 利用跳色哲學，在柔和中建立清晰的視覺層級，符合 Fitts's Law 操作動線。
* **全裝置適應 (RWD)**：無論是桌機的大螢幕寫作，或是手機/平板的分頁模式 (Tabs)，介面皆能完美流體適應。

### 2. 專業級編輯工具列 (Pro Toolbar)
* **視覺明確化**：捨棄晦澀圖示，採用清晰的文字標籤按鈕 (如 H1 24pt, A+, A-)。
* **智慧清空選單**：整合「清空原文」、「清空關鍵字」與「全部重置」，防止誤刪重要資料。
* **無損字級技術**：所有標註操作僅改變顏色與權重，**絕對不影響** 原本設定的字體大小 (Font Size Agnostic)。

### 3. 無縫交付與協作 (Seamless Delivery)
* **一鍵複製到 Docs**：按鈕整合了格式清洗邏輯，直接貼上 Google Docs 即可交付，無需二次排版。
* **跨境發布支援**：內建輕量級 **繁簡轉換引擎**，一鍵切換語系，滿足兩岸三地多平台發布需求。
* **純文字備份**：支援 `.txt` 格式導出，方便原始碼保存。

## 📝 使用指南 (User Guide)

只需簡單步驟，完成一篇標準化的 SEO 文章：

1.  **【入稿】Input**
    * 在左側貼上草稿。視需求點擊「簡→繁」或「繁→簡」進行轉換。
    * 確認「全形標點」已開啟，自動修正符號格式。
2.  **【定義】Define**
    * 在「關鍵字庫」輸入目標 SEO 關鍵字（每行一個）。
    * 點擊醒目的紅色 `一鍵標註` 按鈕，系統自動為您畫上重點。
3.  **【結構】Structure**
    * 選取大標題，點擊上方工具列的 `H1`。
    * 選取副標題，點擊 `H2`。
    * 確保文章結構層級分明，提升易讀性。
4.  **【交付】Export**
    * 點擊工具列中央深藍色的 `複製` 按鈕。
    * 前往 Google Docs 按下貼上 (Ctrl+V)，格式完美呈現。
