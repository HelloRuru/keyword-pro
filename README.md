## 📖 序言 (Introduction)

**蛍光ペン PRO (Highlighter PRO)** 是一款運行於瀏覽器的 **SEO 寫作輔助工具 (SPA)**。

它不僅僅是軟體，更是一種對書寫體驗的致敬。我們結合了傳統日式文具的 **侘寂 (Wabi-sabi)** 美學與現代硬派的 SEO 邏輯，填補了專業寫手在「撰稿、標註、交付」流程中的斷層。

**v9.4 Visual Sync Edition** 搭載了最新的 **「視覺同步」** 與 **「樣式烘焙」** 雙重引擎，確保您在編輯器中看到的每一個像素、每一種字級，都能完美無損地交付至 Google Docs。

## ⚡ 痛點與解方 (Philosophy)

針對 SEO 內容工作者最常遇到的挑戰，我們以「結構化」思維重新設計了解決方案。

| 寫手的痛點 (Pain Points) | 蛍光ペン PRO 的解法 (Solution) |
| :--- | :--- |
| **Google Docs 格式不一** | **Style Baking 技術**<br>複製當下將字體 (12/18/24pt) 與行高直接「烘焙」進標籤，徹底解決 Docs 忽略外部樣式的問題。 |
| **預覽與實作落差** | **Visual Sync 引擎**<br>透過高權重 CSS 強制渲染，編輯當下即是最終結果，所見即所得。 |
| **操作焦點遺失** | **Selection Memory 機制**<br>自動記憶游標位置，防止點擊工具列時焦點遺失，提供如實體書寫般的流暢感。 |
| **手動標註耗時** | **一鍵標註 (Auto Mark)**<br>瞬間將目標關鍵字標示為顯眼的 **[茜色+粗體]**，大幅提升校對效率。 |
| **標題層級混亂** | **語意化結構按鈕**<br>直觀設定 H1/H2/P 標籤，讓文章骨架符合 SEO 搜尋引擎最佳實踐。 |

## ✨ 核心機能 (Craftsmanship)

### 01. 利休色調・沉浸式寫作空間 (Rikyu Workspace)
* **和風紙質感**：以「枯野色」與「潤色」調和出的溫潤背景，疊加極微細的和紙噪點紋理。
* **視覺引導設計**：利用「跳色」哲學——以 **茜色** 進行重點標註、**紺青色** 進行交付確認，在柔和中建立清晰的視覺層級。

### 02. 專業級編輯工具列 (Pro Toolbar)
* **視覺明確化**：捨棄晦澀的圖示，採用清晰的文字標籤按鈕，回歸工具本質。
* **精準字級控制**：支援 10pt-36pt 跨段落調整，修復舊版邊框顯示問題。
* **無損字級技術**：所有標註操作僅改變顏色與權重，**絕對不影響** 原本設定的字體大小與行距。

### 03. 無縫交付與協作 (Seamless Delivery)
* **一鍵精準複製**：按鈕整合了格式清洗與注入邏輯，直接貼上 Google Docs 即可交付，無需二次排版。
* **跨境發布支援**：內建輕量級 **繁簡轉換引擎**，一鍵切換語系，滿足兩岸三地多平台發布需求。

## 🔰 使用指南 (User Manual)

只需簡單 4 步驟，3 分鐘內完成一篇標準化的 SEO 文章：

### Step 1. 入稿 (Input)
> 在左側側邊欄貼上您的文章草稿。
* 如需發布到不同地區，可點擊 `簡→繁` 或 `繁→簡` 轉換。
* 確認 `全形標點` 已勾選，系統將自動優化標點符號。

### Step 2. 定義 (Define)
> 在左下方「關鍵字庫」輸入鎖定的 SEO 關鍵字（每行一個）。
* 輸入後，點擊醒目的紅色 **[一鍵標註]** 按鈕。
* 系統將自動把文內所有關鍵字轉為 **紅色粗體**，並計算密度。

### Step 3. 結構 (Structure)
> 這是 SEO 最重要的一步：設定文章骨架。
* 選取大標題 → 點擊工具列 `H1` (視覺轉為 24pt)。
* 選取副標題 → 點擊工具列 `H2` (視覺轉為 18pt)。
* 如有需要，點擊底部狀態列切換 **「含標點 / 不含標點」** 字數統計。

### Step 4. 交付 (Export)
> 完成編輯，準備交稿。
* 點擊右上方深藍色的 **[複製]** 按鈕 (出現成功提示)。
* 前往 Google Docs 按下貼上 (Ctrl+V)，**字體大小、顏色與格式** 將完美鎖定。
