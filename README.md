## 📖 使用作法 (How to Use)

為了讓您能迅速上手這款「空氣感」編輯器，我們將操作流程拆解為簡單的四個步驟：

### 01. 原稿注入 (Input)
請在左側（手機版為「編輯輸入」分頁）貼上您的文章草稿。
* *Tip:* 系統會自動過濾雜訊，僅保留純粹的文字內容。

### 02. 關鍵字設定 (Keywords)
在左下方的關鍵字欄位輸入您想鎖定的 SEO 字詞（每行一個）。
* **即時觀測**: 右側會立即顯示該關鍵字在文章中的密度（Density）。
* **職人建議**: 將密度控制在 **1% - 2.5%** 之間，是搜尋引擎最喜歡的甜蜜點。

### 03. 一鍵入魂 (Auto Mark)
點擊 <kbd>一鍵標註 (Auto Mark)</kbd>。
系統將自動為所有關鍵字套上「朱赤」色虛線底線與加粗效果，讓重點一目瞭然，卻不破壞閱讀的視覺流動。

### 04. 手感微調與輸出 (Refine & Export)
切換至右側預覽區（手機版為「預覽畫布」分頁）：
* **色彩選用**: 使用上方工具列的「日本傳統色」色票（如櫻色、若苗、藤色）進行螢光筆畫記。
* **重點強調**: 選取文字後點擊 <kbd>A+ 重點</kbd>，施加強烈的視覺引導。
* **最終輸出**: 點擊右上角的 <kbd>複製文案</kbd>，即可帶著完整的 HTML 格式貼入 Google Docs 或 CMS 後台。

## 為什麼我們對線條與色彩如此偏執？

面對繁忙的內容產製需求，很多人最怕的就是標註重點時那種生硬的斷層感。說真的，我們在開發這款工具時，核心理念就是想讓 **SEO 關鍵字標註** 過程變得像是在實體雜誌上編輯一樣直覺。這不只是一個代碼工具，它更像是我們對文字質感的一種堅持。我們把那種細線條的視覺美學藏在每一個 Icon 與邊框裡，希望能陪著在高雄、台南或台北各地的寫手夥伴們，一起找回對文字最純粹的掌控感，讓讀者在每一篇文案中都能讀到一份專屬的生活提案。

身為深耕多年的編輯團隊，我們發現大家在處理關鍵字標註時，版面往往會變得很雜亂。其實，排版不僅僅是為了美觀，它更像是一場無聲的對話。萬一標記太突兀，讀者的閱讀節奏就會被打斷。我們在構思這套工具時，最先考慮的不是功能多強大，而是寫手在編輯時的心情是不是夠放鬆。

那種日式雜誌感的極細線條，其實是為了讓讀者的眼睛在大數據時代能有個喘息的空間。我們不希望大家只是在填空，而是像拿著鋼筆在米紙上劃過的手感，一片片地鎖上文字零件，確保最後呈現出來的文章結構紮實且充滿流動感。

## 🎨 職人色彩規範 (Wairo System)

我們精選了五款最適合長時間閱讀的日本傳統色，作為工具的靈魂：

* **墨 (Sumi)**: `#2C2C2C` —— 取代死板純黑，賦予文字墨水乾透後的質感。
* **朱赤 (Shu-aka)**: `#B93A32` —— 最顯眼的標註，帶有日本傳統色彩的溫潤。
* **藍鐵 (Aitetsu)**: `#374B6D` —— 專業的骨架，支撐起整篇內容的穩重感。
* **常盤 (Tokiwa)** / **深紫 (Kokimurasaki)** : 用於細節層次的區分。
