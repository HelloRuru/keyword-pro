## 📖 序言 (Introduction)

**蛍光ペン PRO (Highlighter PRO)** 不僅僅是一個編輯器，它是運行於瀏覽器上的 **「SEO文案數位工具」**。

我們摒棄了繁雜的後台，回歸書寫本質。透過 **「G 筆尖 (G-Pen Nib)」** 的意象與 **「生成色 (Ecru)」** 紙張紋理，營造出如同翻閱日系雜誌般的視覺體驗。本工具採用 **PWA 架構**，可直接安裝於手機或電腦桌面，實現零延遲的離線寫作體驗。

我們針對 SEO 寫手的真實工作場景，重新定義了數位寫作的標準。

| 寫手的痛點 (Pain Points) | 蛍光ペン PRO 的解法 (Solution) |
| :--- | :--- |
| **Google Docs 貼上跑版** | **樣式烘焙技術 (Style Baking)**<br>獨家開發的轉換引擎，將網頁上的 H2/H3/色票/螢光筆，強制封裝為 Google Docs 可識別的內嵌樣式。**複製即貼上，格式零誤差。** |
| **字數控制困難** | **即時數據儀表 (Live Stats)**<br>底部常駐字數統計條。支援「總字數」與「選取範圍字數」雙重監測，並可一鍵切換「含/不含標點符號」，精準控制 SEO 密度。 |
| **手機修改不便** | **浮動文具盒 (Mobile Float Menu)**<br>專為行動裝置設計。在手機上選取文字時，會自動彈出精簡版工具列（含同義詞提示），讓手指無需在螢幕上下來回奔波。 |
| **缺乏 SEO 靈感** | **邏輯式 AI (Logic-Based AI)**<br>不需串接昂貴 API。內建啟發式演算法，根據您的文章內容與關鍵字，一鍵生成 5 組高點擊標題與 Meta Description 建議。 |

## ✨ 機能工藝 (Craftsmanship)

### 01. 雜誌感介面設計 (Magazine UI)
* **視覺語言**：全站採用 `1.2px` 極細線條 SVG 圖示，搭配 `Shippori Mincho` (襯線體) 與 `Zen Kaku Gothic` (黑體)，呈現日系排版的空氣感。
* **深色模式**：搭載 **「墨金 (Sumi-Gold)」** 夜間主題，配合 **毛玻璃 (Glassmorphism)** 特效工具列，提供沉浸式的夜間寫作體驗。

### 02. 結構化編輯 (Structured Editing)
* **SEO 標籤鎖定**：提供 H2 (章節)、H3 (子題)、P (內文) 獨立按鈕，確保文章結構符合搜尋引擎喜好。
* **文具系配色**：內建 5 款精選 **「日本傳統色」** 文字墨水（如：藍鉄、千歲綠）與 5 款柔和螢光筆，告別刺眼的預設色彩。

### 03. 安全與防護 (Security & Legal)
* **防災記憶 (Auto-Save)**：內建 LocalStorage 引擎，即時監聽每一個按鍵。無論瀏覽器崩潰或斷電，靈感都被完整保留。
* **資料備份**：提供 **[下載備份]** 功能，將草稿導出為實體檔案，資料主權完全掌握在您手中。
* **防盜機制**：底層植入 Layer 1 防護腳本，禁用右鍵與原始碼檢視。

## 🔰 使用指南 (User Manual)

### Step 1. 入稿 (Input)
> 在左側側邊欄貼上您的文章草稿。
* 系統支援純文字貼上，會自動轉換為標準段落格式。
* 點擊 **「✨ AI 智慧建議」**，獲取 SEO 標題與摘要靈感。

### Step 2. 標註 (Highlight)
> 在左下方輸入 SEO 關鍵字。
* 點擊 **[一鍵標註]**，系統將自動計算密度並標示關鍵字。
* 利用滑鼠游標滑過左側關鍵字卡片，右側文章會產生 **呼吸燈效果**。

### Step 3. 編輯 (Edit)
> 像在紙上畫重點一樣。
* 選取文字，透過頂部（或手機浮動）工具列進行 **H2/H3 設定**、**文字上色** 或 **畫螢光筆**。
* 若需修改，使用 **[橡皮擦]** 圖示可一鍵清除格式但保留文字。

### Step 4. 交付 (Delivery)
> 完美的最後一哩路。
* 確認底部字數符合需求。
* 點擊 **[複製到 Docs]**。
* 前往 Google Docs 按下貼上 (Ctrl+V)，格式將完美呈現。
